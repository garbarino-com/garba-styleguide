1.1.3 #Components.CellList.3-CheckboxCellList Cell list con checkboxes

Las Cell-lists pueden usarse dentro para checkboxes dentro de un form para seleccionar entre un set de ítems que puede contener textos o imagenes.

Markup: cell-list/cell-list-with-checkboxes.hbs
