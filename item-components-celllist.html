1.2 #Components.CellList Cell list

Es una adaptación del componente material. Es una alternativa a la lista común. Un grupo de celdas de forma cuadrada organizadas en forma horizontal.

Las Cell-lists sirven se usan para tipos de datos similares. Ayudan a mejorar la comprensión visual de los datos que agrupan.

Los ítems que componen las Cell-lists tienen un ancho fijado que se asigna automáticamente de acuerdo a la cantidad de siblings que hayan en la misma fila (máximo 6).

Los ítems que componen la lista se usan con la clase .cell-list-item salvo cuando se utiliza con imagenes, en cuyo caso se debe utilizer el elemento Image Container.
