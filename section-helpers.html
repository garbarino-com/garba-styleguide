4 #Helpers Helpers

Un helper es un conjunto de declaraciones de CSS que se pueden reutilizar. También se puede usar con parámetros de modo de hacerlo aún más flexible. Usage:

.classname {
  @include mixin-name($params);
}

4.10.1 #Helpers.Utilities.ElementAspectRatio @element-aspect-ratio($width, $height)

Parameters:
  • $width
    Número para indicar la proporción de ancho (Default: 1).
  • $height
    Número para indicar la proporción de alto (Default: 1).
