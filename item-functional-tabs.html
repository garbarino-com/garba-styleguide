3.2 #Functional.Tabs Tabs

Navegación dinámica con transición de contenidos locales. Se puede utilizar en conjunto con el componente Navbar. La funcionalidad de las tabs esta extraída de las tabs de Bootstrap y tiene los mismos métodos y eventos.

En este ejemplo la vemos usada con un Navbar con las variables:

  • Navbar con íconos (.navbar__with-icons).
  • Navbar justificada (.navbar__justified)
  • Home
  • Profile
  • Settings
  • Messages
